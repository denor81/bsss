# Basic Server Security Setup (BSSS)

Скрипт для базовой быстрой и легкой настройки безопасности только что установленного сервера Ubuntu.

## Использование

### One-line запуск
```bash
curl -s https://raw.githubusercontent.com/user/repo/main/launcher.sh | bash -s -- [опции]
```

### Опции командной строки
- `--check` - Режим проверки (без внесения изменений)
- `--default` - Режим отката к настройкам по умолчанию
- `-v` или `--verbose` - Подробный вывод

## Функционал

1. Проверка необходимости перезагрузки системы
2. Обновление системных пакетов
3. Смена SSH порта
4. Отключение IPv6
5. Отключение авторизации по паролю для SSH
6. Настройка брандмауэра UFW

## Требования

- Операционная система: Ubuntu (с поддержкой systemd)
- Права: root или пользователь в группе sudo
