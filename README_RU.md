# 🛡️ BSSS — Basic Server Security Setup

Быстрая базовая защита свежего Ubuntu-сервера одной командой с авто-откатом, если новые SSH-настройки отрежут доступ.

## Назначение

Быстро и безопасно настроить базовые правила:
- Запретить SSH авторизацию по паролю
- Запретить вход пользователя root
- Сменить SSH порт и синхронизировать правила UFW
- Включить UFW
- Опционально создать swap файл

Авто-настройка требует:
- Вход по SSH ключу
- Пользователя не root (sudo)

Также можно настроить каждый модуль через меню.

## Ключевые особенности

- Откат по таймеру (300с) для защиты от потери доступа
- Полное логирование: терминал, файлы, systemd journal
- Модульная архитектура
- Установщик в одну строку с GPG-проверкой

## Совместимость

- Только Ubuntu (проверено 20.04–24.04)
- Используется sshd `Include`, поэтому старые версии не поддерживаются

## Одна команда

```bash
curl -fsSL https://raw.githubusercontent.com/denor81/bsss/main/oneline-runner.sh | sudo bash
```

## Использование

```bash
sudo bsss      # Меню
sudo bsss -u   # Удалить
```

## Логи

```bash
journalctl -t bsss --since "10 minutes ago"
```
